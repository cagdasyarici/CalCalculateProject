# CalCalculateProject
Calorie Calculator Application.MSSQL database was used with the CodeFirst approach to create the database.One-to-many and many-to-many relationships were implemented.Seed data is available.There are two different user types:admin and regular user, with different panels.The UI was created using Windows Forms,and it includes a sidebar and an MdiFom.
